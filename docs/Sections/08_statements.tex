\section*{Reproducibility Statement}

Algorithm~\ref{alg:loss-indexed-confidence} specifies the score;
Appendix~\ref{app:theory} proves the bounds; Sections~\ref{sec:experiments}
and~\ref{app:benchmark-protocol} freeze the protocol. The anonymous supplement
provides hash-validated per-image records, tie-aware analysis code, generated
tables, and cross-file verification.

\section*{Ethics Statement}

We create no patient data and use only released, de-identified benchmarks.
Dataset and annotation biases can limit rankings; abstention is neither clinical
validation nor a safety guarantee.
