\section*{Reproducibility Statement}

Algorithm~\ref{alg:loss-indexed-confidence} specifies the score;
Appendix~\ref{app:theory} proves the bounds; Sections~\ref{sec:experiments}
and~\ref{app:benchmark-protocol} freeze cohorts, actions, losses, comparators,
and empty-mask rules.  The supplement provides one-job-per-experiment scripts,
hash-validated manifests, tie-aware analysis, generated tables, and tests.

\section*{Ethics Statement}

We create no patient data and use only released, de-identified research
benchmarks.  Device, acquisition, demographic, and annotation biases can limit
confidence rankings; abstention is not clinical validation, a safety guarantee,
or a substitute for domain-expert review.
